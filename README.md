# todo-backend-nodejs
Todo app backend in Node.js &amp; express.js

## System Requirements
Node: `v12.16.1`
Postgres: `10.10`

## Step to setup project
1. Clone the porject on your local machine
2. run `npm i -g db-migrate`
3. CD into the prject folder and run `npm install`
4. Open `database.js` file and update the credentials that you are using for postgres database.
5. run `db-migrate up`

## run server
`node app.js`
